# Duodecimfolium Release Notes

CurrentVersion: 0.3.6
CurrentDate: 2026-03-09

## 0.3.6 | 2026-03-09
- 修复时间轴 UI 显示范围，统一为 `0-12`（共 13 个刻度）。
- 修复【超光速】触发【超级动能撞击】时速度取值错误：改为使用被触发卡实例的当前速度。
- 新增临时卡包“神人小妙招”，设计师：橘猫（售价 10 金币 + 1 枚【黑手】）。
- 新增地牢“粪怪地带”，设计师：橘猫（解锁条件：通关“初心剑馆”，通关奖励 20 金币）。
- 新增“神人小妙招”卡牌脚本与对应敌方配置。

## 0.3.5 | 2026-03-07
- 新增“设计师”字段，支持在卡牌与地牢界面显示署名（无字段则不显示）。
- 新增地牢“变速齿轮，启动！”，设计师：木头
- 新增临时卡包“变速齿轮组”，仅在持有信物【变速齿轮】时显示并可购买。
- 新增【变速齿轮】信物及对应掉落：精英 50% 掉落 1 枚，Boss 100% 掉落 1-2 枚。
- 新增“变速齿轮组”12 张卡及对应战斗脚本实现。
- 新增装备“高速引擎”“超载装货”的战斗内持续效果联动。

## 0.3.1 | 2026-03-07
- 修复回放结算异常与重复掉落问题。
- 调整拉克希尔之仪与多段伤害相关规则。
- 新增 maxCopies 卡牌上限规则并同步 UI 提示。
- 卡牌长文本改为摘要展示，支持悬停查看完整说明。
- 首页版本信息改为单文件维护，便于设计手动更新。
