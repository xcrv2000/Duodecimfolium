新机制：信物
信物是一类特殊的掉落物，和修饰珠并列。
有一些卡包在进行购买时需要消耗对应数量和种类的信物。
卡包应当标记自己需要消耗哪种信物。当鼠标悬停在信物上时，告诉玩家该信物由哪个地牢产出。

新机制：【全力】
全力是一个卡可以拥有的关键词。拥有该关键词的卡自动拥有【全力】tag。
拥有【全力】关键词的卡具有以下效果：
    如果该角色在该tick发动过其它的卡（实例），该卡无法发动（这张卡的结算被跳过。这意味着这张卡必须是这个角色在该tick发动的第一张卡才能正常发动）。
    成功发动该卡后，该tick中所有属于该角色，且未发动过的卡速度+1（speed10 + 10，意味着剩余的卡都会被延迟到下一tick）
这意味着一个tick只能打出一张拥有【全力】的卡。

为卡包2的【气定神闲】增加【全力】关键词。


新机制：【护具】
【护具】是一个卡上的关键词。持有该关键词会自动获得【护具】tag
玩家只能在卡组中携带一张【护具】
护甲一般没有速度，在战斗开始时发动，获得一个全局、持续至战斗结束的角色buff。
