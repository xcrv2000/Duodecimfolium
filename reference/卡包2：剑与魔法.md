# 卡包2：剑与魔法

1. 火球

速度：6

标签：攻击 / 魔法 / 火

稀有度：6

效果：
对随机敌方单位造成 6 点魔法火焰伤害。
随后，对其余敌方单位各造成 3 点魔法火焰伤害。

2. 冰球（暂名）

速度：6

标签：攻击 / 魔法 / 冰；攻击 / 物理

稀有度：6

效果：
对随机敌方单位造成 6 点伤害（该伤害同时具有“魔法/冰”与“物理”标签）。
随后，使敌方单位在本回合后续所有“攻击类卡”的速度获得 -1 的卡类速度 buff，持续至回合结束，并立即重排时间轴。

说明：

“后续所有攻击类卡”指：在效果结算后，敌方仍未触发的、且带有标签“攻击”的卡实例。

已经触发过的攻击不受影响。

3. 石球（暂名）

速度：6

标签：攻击 / 物理

稀有度：6

效果：
对随机敌方单位进行三次攻击。每次造成 2 点物理伤害。
每次伤害独立结算（护甲抵扣与相关 buff 分别触发）。

4. 清亮剑油

速度：4

标签：无

稀有度：6
效果：
使自身获得“清亮剑油”状态（不可叠加，持续至战斗结束）。

清亮剑油状态效果：
当自身下一次发动带有标签“攻击/物理”的卡牌时：

在该次攻击结算完成后，为该卡牌对应的卡定义施加一个“永久附着（到战斗结束）”的速度修正：从下回合开始，该卡的所有后续卡实例获得 速度 -2（速度最小不低于 1），并在生效时立即重排时间轴。

该状态随后移除（仅触发一次）。

实现注记：

这里的“从下回合开始”意味着：如果该攻击在本回合已生成的卡实例里还有后续实例，不立刻改；从下一回合生成/调度该卡实例时开始应用。

“不可叠加”指同一单位身上该状态为 non-stackable。

7. 威仪

威仪标记效果：

9. 回旋斩

速度：4

标签：攻击 / 物理

稀有度：6

效果：
对随机敌方单位造成 3 点物理伤害。
若该次伤害结算后，目标单位仍存活，则向自身卡组中衍生一张【回旋·斩】。

实现注记：
你草稿里写了“命中且未被格挡”，但当前系统未定义“格挡”。本规范暂用“目标仍存活”作为触发条件占位；若你后续加入格挡系统，把条件替换即可。

13. 气定神闲

速度：5

标签：无

稀有度：6

效果：
使自身获得“气定神闲”状态（不可叠加，持续至回合结束）。

气定神闲状态效果：
non-stackable，层数恒定为1

当自身下一次发动带有标签“攻击/魔法”的卡牌时：

该攻击的伤害结算重复执行 5 次（即视为 5 次独立的伤害结算）。

随后，移除该攻击卡（卡工厂），并移除提供该buff的【气定神闲】卡（卡工厂）。

该状态触发后移除。


14. 风雷击

速度：6

标签：攻击 / 物理

稀有度：6

效果：
对随机敌方单位造成 6 点物理伤害。
随后，使本卡获得“风雷余势”（可叠加，持续至战斗结束）。

风雷余势效果：
该卡在后续生成的卡实例中获得 速度 -1（最小不低于 1）。
每层风雷余势使上述速度修正额外 -1。

实现注记：
为避免“改到过去 tick 后立即二次触发”的复杂性，建议该速度修正仅作用于“后续回合生成/调度的该卡实例”（类似剑油）。

15. 便携干粮（补给品）

速度：无（非战斗 tick 卡）

标签：补给品

稀有度：6

携带限制：
一个卡组中最多携带 3 张“补给品”标签卡（全局限制，非本卡专属）。

效果：
在战斗结束时触发（不进入 tick 调度）：
若自身 hp / maxHp < 70%，则回复 20 点生命。

该卡具有“可触发次数”计数：最多触发 3 次。触发次数用尽后，该卡失效（不再触发）。
